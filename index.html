GitHub 지식 마인드맵

GitHub 레포지토리의 실제 폴더 구조를 읽어서 바로 마인드맵으로 보여줍니다. 즉, GitHub에서 폴더를 바꾸면 화면도 그 구조를 반영합니다.

레포지토리 연결
아직 GitHub 구조를 불러오지 않았습니다.
로딩 전
레포지토리를 연결하면 폴더 구조가 마인드맵으로 표시됩니다.
폴더 하나 = 지식 덩어리 하나
하위 폴더 = 하위 개념
README.md = 정리 자료 링크
GitHub 구조를 다시 읽을 때마다 자동 반영
드래그: 화면 이동
휠: 확대 / 축소
노드 클릭: 상세 보기
GitHub API 기반 자동 구조 반영
GitHub Pages에 index.html로 배포 가능
owner / repo / branch 지정 후 자동 시각화
